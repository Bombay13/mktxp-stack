## Description
MKTXP-Monitor is an out-of-the-box dockerized monitoring stack for [MKTXP Exporter](https://github.com/akpw/mktxp). This is mostly a complimentary project to get people up & running with mktxp, though it does add some extra capabilitis such as Mikrotik router log monitoring, etc. 


#### Requirements:
[Docker](https://docs.docker.com/get-docker/)
[Docker Compose](https://docs.docker.com/compose/install/)
